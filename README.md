# EE-video-from-CSV
Sample application to pull in a list of videos from matching timestamps in a CSV

### How to Install ###

1: `do stuff`

2: `create an 'uploads' directory`

### How to Run ###

1: `export FLASK_APP=hello.py`

2: `export FLASK_ENV=development`

3: `flask run`


### Format for Excel file ###

1: `example.xlsx`
